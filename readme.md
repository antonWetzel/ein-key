Simple Key-Remapper for Windows

## Usage

- Create or edit remapping
	- Click left box and press wanted input
	- Click right box and press wanted output
	- Confirm
- Configuration
	- Export to save configuration as JSON
	- Import to load from saved JSON
- Stop remapping
	- Close program
